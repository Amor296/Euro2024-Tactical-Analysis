## üìä Key Features
* **Dynamic Visualizations:** Generate Pass Maps, Heatmaps, and Carry Trajectories on the fly.

* **Performance Metrics:** Real-time calculation of Pass Accuracy, Total Passes, and Expected Threat (xT).
